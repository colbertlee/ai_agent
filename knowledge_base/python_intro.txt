Python Introduction

1. What is Python?
Python is a high-level, general-purpose programming language. Created by Guido van Rossum in 1991.

2. Features
- Easy to learn
- Object-oriented
- Dynamic typing
- Cross-platform
- Rich libraries

3. Applications
- Web development (Django, Flask)
- Data analysis (NumPy, Pandas)
- AI/ML (TensorFlow, PyTorch)
- Automation
